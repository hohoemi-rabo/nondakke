# 08 — カテゴリ切替タブ

> **変更履歴**：当初は左右スワイプ＋ドットインジケーターで実装したが、視認性・発見しやすさの問題からタブUIに変更した（2026-07）。スワイプジェスチャーと循環ロジック（cycleCategoryFilter）は削除済み。

## 概要

カレンダー画面上部のタブで表示カテゴリを切り替える。「全部」／「お薬」／「サプリ」／「その他」。

## 参照

- REQUIREMENTS.md §5.1（タブでカテゴリ切替）
- DESIGN.md §5（カテゴリ切替タブ）

## タスク

- [x] カレンダー画面上部にカテゴリタブを実装（`components/calendar/category-tabs.tsx`）
  - 全部／お薬／サプリ／その他 の4等分タブ。選択スタイルは登録フォームのチップと共通
- [x] カテゴリでフィルタした状態をカレンダー・サマリーカードの両方に反映
  - 「全部」以外では該当カテゴリのアイテムの予定・記録のみ表示
- [x] 切替アニメーションは控えめに（DESIGN.md「アニメーション過多」禁止。フェード程度）

## 設計メモ

- フィルタの状態管理（`lib/category-filter.ts` の `CategoryFilter` / `CATEGORY_FILTER_ORDER`）は切替UIと分離してある
- タブ化により月送り・縦スクロールとのジェスチャー競合の考慮は不要になった

## 完了条件

- タブでカテゴリが切り替わり、カレンダーのドット・サマリー数字がフィルタされる
