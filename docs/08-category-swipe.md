# 08 — カテゴリ切替スワイプ

## 概要

カレンダー画面を左右スワイプして表示カテゴリを切り替える。「全部」→「お薬」→「サプリ」→「その他」のループ。

## 参照

- REQUIREMENTS.md §5.1（左右スワイプでカテゴリ切替）
- DESIGN.md §5（ドットインジケーター＋ラベル）

## タスク

- [ ] カレンダー画面に左右スワイプジェスチャーを実装（react-native-gesture-handler は導入済み）
  - 順序：全部 → お薬 → サプリ → その他 → 全部（ループ）
  - 左右どちらのスワイプでも逆順に一周できる
- [ ] 現在カテゴリのインジケーターを画面上部に表示
  - ドット4つ＋現在カテゴリのラベル（例：「お薬」）
  - 選択中ドット = accent、非選択 = textMuted
- [ ] カテゴリでフィルタした状態をカレンダー・サマリーカードの両方に反映
  - 「全部」以外では該当カテゴリのアイテムの予定・記録のみ表示
- [ ] 切替アニメーションは控えめに（DESIGN.md「アニメーション過多」禁止。スライドまたはフェード程度）

## 設計メモ

- 月送り操作（前月/翌月）とスワイプが競合しないこと。月送りはボタン、カテゴリ切替はスワイプに役割分担する
- Phase 2 でタブUIへの変更候補（REQUIREMENTS.md §8）。カテゴリの状態管理は切替UIと分離して実装しておく

## 完了条件

- スワイプでカテゴリが循環切替され、カレンダーのドット・サマリー数字がフィルタされる
- 縦スクロールや日セルタップとジェスチャーが干渉しない
