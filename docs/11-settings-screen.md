# 11 — 設定画面

## 概要

3番目のタブ。データ全削除・アプリ情報・プライバシーポリシー表示。ストア公開要件を満たすための画面。

## 参照

- REQUIREMENTS.md §5.5（設定画面）、§7（非機能要件）、§9（スコープ外：医療アドバイスを行わない旨の明記）

## タスク

- [ ] `app/(tabs)/settings.tsx` に設定リストを実装
- [ ] データ全削除
  - 確認ダイアログを挟む（記録ボタンと違い、ここは破壊的操作のため確認必須）
  - items / intake_records を全削除（トランザクションで実行）
- [ ] アプリ情報：アプリ名・バージョン（expo-constants から取得）
- [ ] プライバシーポリシー表示
  - 内容：データはすべて端末内に保存、ネットワーク送信なし、収集情報なし
  - アプリ内表示（静的テキスト画面）とする。外部URL方式にする場合はGoogle Play要件（公開URL必須）を12で対応
- [ ] 免責事項の明記：本アプリは記録ツールであり医療アドバイスを行わない旨（REQUIREMENTS.md §9）

## 設計メモ

- 通知設定・テーマ切替などは置かない（MVPに存在しない機能の設定項目を作らない）

## 完了条件

- データ全削除後、カレンダー・一覧が空状態になる
- プライバシーポリシーと免責事項が閲覧できる
