# DESIGN.md — のんだっけ？

服薬・摂取記録カレンダーアプリ「のんだっけ？」のデザインシステム定義。
REQUIREMENTS.md とセットで使用する。実装は Expo (React Native) 前提のため、値はすべて RN の StyleSheet で使える形で定義する。

## 1. デザイン原則

1. **白が主役**。色は意味があるときだけ使う（面積比の目安＝白9割：色1割）
2. **装飾レス**。グラデーション・影の多用・イラスト背景は使わない。余白とタイポ階層で見せる
3. **医療アプリ然としない**。青・赤の医療系配色、十字マーク、錠剤アイコンの多用を避ける
4. **1画面1メッセージ**。各画面で一番伝えたい情報（今日のむもの／カレンダー）以外は脇役に徹する
5. **タップの気持ちよさ**。記録ボタンは大きく、押した瞬間のフィードバック（チェック表示）を必ず返す

## 2. カラートークン

```ts
export const colors = {
  // ベース
  background: '#FFFFFF',      // 画面背景（全画面共通）
  surface:    '#FFFFFF',      // カード背景
  border:     '#F1EFE8',      // 区切り線（0.5px相当の薄さで使う）

  // テキスト
  textPrimary:   '#2C2C2A',   // 本文・日付数字
  textSecondary: '#5F5E5A',   // 補足・凡例
  textMuted:     '#B4B2A9',   // 非活性・過去日

  // アクセント（ティール）
  accent:      '#1D9E75',     // ドット・チェック・選択中タブ
  accentDeep:  '#0F6E56',     // アクセント上の小テキスト
  accentDark:  '#04342C',     // アクセント淡背景上の見出し・数字
  accentLight: '#E1F5EE',     // サマリーカード背景・今日セルのハイライト

  // カテゴリ色
  catMedicine:   '#1D9E75',   // お薬（アクセントと共通）
  catSupplement: '#D85A30',   // サプリ（コーラル）
  catOther:      '#B4B2A9',   // その他（グレー）

  // カテゴリ淡色トーン（選択中タブ・サマリーカード・今日セルの背景とラベル）
  catMedicineLight:   '#C5EADB', // お薬選択時の背景（「全部」の accentLight より一段濃いミント）
  catSupplementLight: '#F9E7DF', // サプリ選択時の背景
  catSupplementDark:  '#7E3418', // サプリ淡背景上のラベル
  catOtherLight:      '#F0EFEB', // その他選択時の背景（ラベルは textSecondary を共用）

  // 曜日
  sunday:   '#993C1D',
  saturday: '#185FA5',
};
```

- **禁止**：上記以外の色の追加（特に赤系のエラー色以外での赤、紫、ピンク）。エラー表示が必要な場合のみ `#C4442A` を使用可
- 濃いティール3種（accent / accentDeep / accentDark）は役割が違うだけの同系色。新しい緑を発明しない

## 3. タイポグラフィ

- フォント：システムフォント（Android = Noto Sans JP / Roboto）。カスタムフォントは導入しない
- ウェイトは **400（regular）と 500（medium）の2つだけ**。boldは使わない

| 用途 | サイズ | ウェイト | 色 |
|------|--------|----------|-----|
| サマリーの大数字 | 26 | 500 | accentDark |
| 画面タイトル・アプリ名 | 17 | 500 | accentDark |
| 月表示（2026年7月） | 14 | 500 | textPrimary |
| 本文・リスト項目 | 14 | 400 | textPrimary |
| カレンダー日付 | 12 | 400（今日のみ500） | textPrimary |
| 補足・凡例・タブラベル | 10–11 | 400 | textSecondary |

## 4. 余白・形状

```ts
export const spacing = { xs: 4, sm: 8, md: 14, lg: 18, xl: 24 };
export const radius  = { card: 14, sheet: 20, chip: 999, cell: 8 };
```

- 画面の左右パディング：14
- カード内パディング：14–16
- タップ領域：最低 44×44。見た目が小さい要素（日セルのドット等）もタップ判定は44以上確保
- 影：カードには使わない。ボトムシートのみ弱い影可（elevation 2相当まで）

## 5. コンポーネント仕様

### カテゴリ切替タブ（カレンダー画面上部）
- 全部／お薬／サプリ／その他 の4等分タブ。角丸 chip、タップ領域44以上
- カテゴリタブには凡例と同じカテゴリ色ドット（8px）を常時表示（全部にはなし）
- 選択中 = カテゴリ淡色背景＋カテゴリ濃色文字（500）で「いまどのカテゴリか」を色でも伝える
  - 全部 = accentLight／accentDark、お薬 = catMedicineLight／accentDark（全部より濃いミント）、サプリ = catSupplementLight／catSupplementDark、その他 = catOtherLight／textSecondary
- 非選択 = 白背景＋border枠＋textSecondary

### サマリーカード（カレンダー画面上部）
- 背景は選択中カテゴリのテーマ淡色（全部=accentLight、お薬=catMedicineLight、サプリ=catSupplementLight、その他=catOtherLight）。角丸 card、影なし
- 左：ラベル「今日のむもの」（11 / テーマ deep 色）＋大数字（26 / テーマ dark 色）
- 右：「✓ のんだ n」「のこり n」（12 / テーマ deep 色）
- 全部飲み終えたら文言を「今日はぜんぶのんだ！」に変える（数字ゼロを見せない）

### カレンダー
- 曜日ヘッダー：日=sunday色、土=saturday色、平日=textSecondary
- 日セル：予定ありの日はカテゴリ色ドット（5px、最大3個、4個以上は「+」表現）
- 記録済みの過去日：ドットの代わりにチェックマーク（accent）、日付数字は textMuted
- 予定があったのに未記録の過去日：ドットを輪郭のみ（塗りなし）で表示
- 今日：セル背景は選択中カテゴリのテーマ淡色（タブ・サマリーカードと連動）・角丸 cell、日付数字 500
- 凡例（お薬・サプリ・その他）をカレンダー直下に常時表示

### 記録ボタン（日別詳細シート内）
- 角丸カード型の大ボタン。ラベル「のんだ！」＋チェックアイコン
- 未記録＝白背景＋border枠、記録済み＝accentLight背景＋accentのチェック
- タップで即時に見た目が切り替わる（確認ダイアログは出さない。取り消しは再タップ）

### ボトムタブ
- 3タブ固定：カレンダー／一覧／設定
- 選択中 = accentDeep、非選択 = textMuted。アイコン20＋ラベル10

### 登録フォーム
- 必須2項目（名前・服用パターン）を最上部に。任意項目は下に寄せる
- 名前のプレースホルダー：「例：血圧の薬、ビタミンD など自分がわかる名前でOK」
- 服用パターンは4択のチップ選択（毎日／N日に1回／曜日指定／不定期）。選択中チップ = accentLight背景＋accentDark文字

## 6. Do / Don't

- Do：迷ったら要素を減らす。空白を恐れない
- Do：状態変化（記録した・し忘れてる）は色とアイコンの両方で伝える
- Don't：バッジ・通知風の赤丸（通知なしアプリの世界観と矛盾する）
- Don't：おめでとう系のモーダル・アニメーション過多（記録は3秒で終わる体験にする）
- Don't：ダークモード対応（MVPではライトのみ。Phase 2判断）
